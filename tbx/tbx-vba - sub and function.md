## Function

**<u>Syntax</u>**

> [**Public** | **Private** | **Friend**] [ **Static** ] **Function** *name* [ ( *arglist* ) ] [ **As** *type* ]
> [ *statements* ]
> [ *name* **=** *expression* ]
> [ **Exit Function** ]
> [ *statements* ]
> [ *name* **=** *expression* ]
> **End Function**
